## Setup
1. Create a Python virtual environment:
    ```sh
    python3 -m venv ~/.udacity-cloud-devops-c02-final
    ```

1. Activate the virtual environment:
    ```sh
    source ~/.udacity-cloud-devops-c02-final/bin/activate
    ```
